# SkillBridge

Đồ án cơ sở ngành

Thành viên:
- Nguyễn Đặng Phương Vy
- Huỳnh Thị Ngọc Anh

Công nghệ sử dụng:
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- SignalR

Chức năng:
- Đăng ký / Đăng nhập
- Học viên / Giáo viên / Admin
- Đặt lịch học
- Chat thời gian thực
- Thanh toán và ví điện tử
- Khiếu nại và xử lý khiếu nại
- Đánh giá giáo viên
- Quản lý giao dịch
